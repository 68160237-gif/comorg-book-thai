# จากที่พวกผมสรุปเนื้อหามาเเล้วจึงได้สร้างข้อสอบ20ชุดขึ้นมาดังนี้
**คำชี้แจง:** เลือกคำตอบที่ถูกต้องที่สุดเพียงข้อเดียว

## **ส่วนที่ 1: พื้นฐานและวิวัฒนาการ**

**1. คำว่า "Core" ที่ใช้เรียกหน่วยความจำหลักในปัจจุบัน มีที่มาจากเทคโนโลยีใดในอดีต?**
- ก. ชิปสารกึ่งตัวนำ
- ข. ห่วงเฟอร์โรแมกเนติกรูปโดนัท
- ค. หลอดสุญญากาศ
- ง. แถบแม่เหล็กความเร็วสูง

**2. คุณสมบัติพื้นฐาน 3 ประการของ Memory Cell ข้อใดกล่าว "ผิด"?**
- ก. มี 2 สถานะที่คงที่ (0 และ 1)
- ข. สามารถเขียนข้อมูลลงไปได้
- ค. สามารถอ่านข้อมูลออกมาได้
- ง. ต้องทำการ Refresh ประจุทุกครั้งหลังการอ่าน

**3. ขั้วสัญญาณใดทำหน้าที่ระบุตำแหน่งของเซลล์เพื่อทำการอ่านหรือเขียนข้อมูล?**
- ก. Data Terminal
- ข. Control Terminal
- ค. Select Terminal
- ง. Vcc Terminal

---

## **ส่วนที่ 2: SRAM vs DRAM**

**4. ข้อใดคือลักษณะเด่นของ SRAM ที่แตกต่างจาก DRAM?**
- ก. ใช้ตัวเก็บประจุในการเก็บข้อมูล
- ข. มีความหนาแน่นของข้อมูลสูงกว่า DRAM
- ค. ทำงานได้เร็วมากและไม่ต้องทำการ Refresh
- ง. ราคาต่อความจุถูกกว่า DRAM

**5. ทำไม DRAM ถึงต้องมีการ "Refresh" ข้อมูลตลอดเวลา?**
- ก. เพื่อป้องกันความร้อนสะสม
- ข. เพราะประจุไฟฟ้าในตัวเก็บประจุมีการรั่วไหล
- ค. เพื่อเปลี่ยนสถานะจาก 0 เป็น 1
- ง. เพื่อล้างข้อมูลเก่าก่อนเขียนข้อมูลใหม่

**6. หน่วยความจำประเภทใดนิยมนำมาใช้ทำเป็น Cache Memory?**
- ก. DRAM
- ข. SRAM
- ค. ROM
- ง. Flash Memory

---

## **ส่วนที่ 3: ROM และ Flash Memory**

**7. ข้อใดอธิบายคุณสมบัติของ "Nonvolatile" ได้ถูกต้องที่สุด?**
- ก. ข้อมูลจะหายไปทันทีเมื่อปิดไฟ
- ข. สามารถอ่านและเขียนข้อมูลได้เร็วเท่ากัน
- ค. ข้อมูลยังคงอยู่แม้ไม่มีกระแสไฟฟ้าเลี้ยง
- ง. ต้องใช้รังสี UV ในการลบข้อมูลเท่านั้น

**8. หากต้องการแก้ไขข้อมูลในหน่วยความจำโดยไม่ต้องถอดชิปออกและทำได้ในระดับ Byte ควรเลือกใช้เทคโนโลยีใด?**
- ก. ROM
- ข. EPROM
- ค. EEPROM
- ง. PROM

**9. Flash Memory พัฒนาขึ้นโดยนำจุดเด่นของหน่วยความจำคู่ใดมารวมกัน?**
- ก. RAM และ ROM
- ข. EPROM และ EEPROM
- ค. SRAM และ DRAM
- ง. MRAM และ PRAM

**10. NAND Flash แตกต่างจาก NOR Flash อย่างไร?**
- ก. NAND อ่านไวกว่าแต่เขียนช้ากว่า
- ข. NOR เหมาะสำหรับทำ SSD มากกว่า NAND
- ค. NAND นิยมใช้เก็บไฟล์ขนาดใหญ่เพราะความหนาแน่นสูงและราคาถูก
- ง. NOR ลบข้อมูลเป็นบล็อก แต่ NAND ลบทีละบิต

## **ส่วนที่ 4: โครงสร้างชิปและการจัดการข้อมูล**

**11. "Multiplexed Addressing" ในชิป DRAM มีวัตถุประสงค์หลักเพื่ออะไร?**
- ก. เพิ่มความเร็วในการอ่านข้อมูล
- ข. ลดจำนวนขา (Pins) บนตัวชิป
- ค. ทำให้ชิปสามารถเก็บข้อมูลแบบ Nonvolatile ได้
- ง. ป้องกันข้อผิดพลาดแบบ Soft Error

**12. หากเพิ่มขา Address บนชิป DRAM อีก 1 ขา (ในระบบ Multiplexed) จะเพิ่มความจุได้กี่เท่า?**
- ก. 1 เท่า
- ข. 2 เท่า
- ค. 4 เท่า
- ง. 8 เท่า

**13. "Interleaved Memory" ช่วยเพิ่มประสิทธิภาพของระบบอย่างไร?**
- ก. ช่วยให้เก็บข้อมูลได้ถาวรมากขึ้น
- ข. ช่วยให้เข้าถึงข้อมูลในหลาย Bank ได้พร้อมกันแบบขนาน
- ค. ลดการใช้พลังงานของหน่วยความจำ
- ง. แก้ไขข้อผิดพลาดแบบ Hard Failure ได้โดยอัตโนมัติ

## **ส่วนที่ 5: การตรวจจับข้อผิดพลาดและเทคโนโลยีใหม่**

**14. "Soft Error" ในหน่วยความจำเกิดจากสาเหตุใดเป็นหลัก?**
- ก. ตัวชิปฮาร์ดแวร์พังเสียหายถาวร
- ข. สัญญาณรบกวนทางไฟฟ้าหรือรังสีคอสมิก
- ค. การลืม Refresh ข้อมูลใน SRAM
- ง. การใช้แรงดันไฟฟ้าสูงเกินไปในตอนเขียน

**15. มาตรฐาน SEC-DED (Hamming Code) มีความสามารถอย่างไร?**
- ก. ตรวจจับได้ 1 บิต และแก้ไขไม่ได้เลย
- ข. แก้ไขได้ 1 บิต และตรวจจับข้อผิดพลาดได้ 2 บิต
- ค. แก้ไขได้ทุกบิตที่ผิดพลาด
- ง. ป้องกันไม่ให้เกิดข้อผิดพลาดได้ 100%

**16. เงื่อนไขในการคำนวณจำนวน Check bits ($K$) เพื่อแก้ไขข้อผิดพลาด 1 บิตในข้อมูล $M$ บิต คือข้อใด?**
- ก. $2^K \ge M$
- ข. $2^K - 1 \ge M + K$
- ค. $M+K = 2^K$
- ง. $K = M^2$

**17. กลไก "Dual-Edge Triggering" ใน DDR SDRAM หมายถึงอะไร?**
- ก. การรับส่งข้อมูล 2 แชนเนลพร้อมกัน
- ข. การรับส่งข้อมูลทั้งขาขึ้นและขาลงของสัญญาณนาฬิกา
- ค. การมีสำรองข้อมูลไว้ 2 ชุดเพื่อกันพลาด
- ง. การใช้ไฟเลี้ยง 2 ระดับ (High/Low)

**18. DDR5 มีการนำนวัตกรรมใดมาใช้เพื่อช่วยให้ระบบเสถียรขึ้นแม้จะมีความเร็วสูงมาก?**
- ก. การลบข้อมูลด้วย UV
- ข. On-die ECC
- ค. การตัดขา RAS/CAS ออก
- ง. การกลับไปใช้ Magnetic Core

**19. เทคโนโลยี MRAM ใช้กลไกใดในการเก็บข้อมูล?**
- ก. ประจุไฟฟ้าในตัวเก็บประจุ
- ข. ทิศทางของแม่เหล็ก
- ค. การเปลี่ยนสถานะจากแก้วเป็นผลึก
- ง. การสร้างสะพานนำไฟฟ้าในชั้นฉนวน

**20. เทคโนโลยี 3D XPoint จัดอยู่ในกลุ่มหน่วยความจำประเภทใด?**
- ก. MRAM
- ข. ReRAM
- ค. PRAM (Phase Change RAM)
- ง. SRAM
