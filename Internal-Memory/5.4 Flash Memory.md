# Flash Memory

**Flash Memory** คือ ลูกผสมที่มาเป็นท็อปของยุคนี้ครับ พัฒนามาจาก EPROM กับ EEPROM รายชื่อจุดเด่นคือ **"ข้อมูลลบไวเหมือนริ้วรอยตา"** สมชื่อ Flash เนื่องจากใช้วิธีลบข้อมูลเป็นบล็อกๆ (Block) หลายครั้งบ่อยครั้งที่อาจจะลบทีละจุดให้เสียเวลา และที่สำคัญคือมัน **Non-volatile** หรือ "ความทรงจำดีไม่มีลืม" แม้ถอดปลั๊กไฟออกสำรวจยังไม่อยู่ครับ

## มันทำงานยังไง
1. **สถานะเลข 1:** คือส่วนที่เกตลอยมัน "ส่วนที่เหลือ" ไม่มีการรองรับใน
2. **สถานะของเลข 0:** บางครั้งใช้ในส่วนของไฟสูงๆ ดันให้อิเล็กตรอนวิ่งทะลุชั้นใดๆ เลยติดแหง็กอยู่ในเกตลอย เรียกว่าคำสั่ง **Tunneling** พอมันเท่านั้นติดแล้วต่อให้ไม่มีไฟเลี้ยงเลยออกไปไหนไม่ได้ ข้อมูลเลยไม่หายครับ
3. **ดูดี:** มันใช้แค่เพียงตัวเดียวต่อ 1 ความเท่าเดิมเลยให้ความสามารถรองรับได้มากกว่าแบบอื่น ๆ อีกมากมาย

## ดังนั้นผมแยกเป็น NOR vs NAND ให้เข้าใจ

### 1. NOR Flash
* **การเชื่อมต่อ:** ต่อแบบขนาน (Parallel) เช่นเดียวกับถนนหลายเลน
* **จุดเด่น:** อ่านข้อมูลไวจัด และคุณรู้ข้อมูลต่อเนื่องกันได้เลย **Random Access** และคุณสามารถรันโปรแกรมจากตัวมันได้เลย แล้วค่อยก๊อปไปลง RAM 
* **ใช้ทำอะไร:** งานภายในเครื่อง เช่น เก็บ FW/BIOS ของพวกไมโครคอนโทรลเลอร์

### 2. NAND Flash
* **การเชื่อมต่อ:** การควบคุมแบบต่อเนื่อง (Series) ประหยัดพื้นที่กว่า
* **จุดเด่น:** เขียนข้อมูลและลบข้อมูลได้ไวมากแถมราคาต่อความจุคือถูกสุดๆ แต่ข้อเสียคือมันอ่านช้ากว่า NOR และต้องอ่าน/เขียนเป็นก้อนๆ (Block/Page) ตรวจสอบเจาะลึกทีละอย่างไม่ได้
* **ใช้ทำอะไร:** เน้นไปที่หนักๆ External Storage ที่ต้องใช้กันทุกวันอย่าง **USB Flash Drive, SD Card** ในมือถือ และ **SSD** ในคอมพิวเตอร์นั่นเอง
