# สรุปเรื่อง Newer Nonvolatile Solid-State Memory Technologies

เทคโนโลยีเหล่านี้ถูกพัฒนาขึ้นเพื่อเป็น **Storage Class Memory (SCM)** โดยมีเป้าหมายที่จะรวมความเร็วของ DRAM และความสามารถในการเก็บข้อมูลถาวรของ NAND Flash เข้าด้วยกัน

---


### 1. Magnetic RAM (MRAM)
ใช้การเก็บข้อมูลด้วย **ทิศทางของแม่เหล็ก** แทนการใช้ประจุไฟฟ้า
* **กลไก:** ใช้โครงสร้าง *Magnetic Tunnel Junction (MTJ)* โดยความต้านทานจะเปลี่ยนตามทิศทางแม่เหล็ก
* **จุดเด่น:** * ความเร็วใกล้เคียง DRAM
    * ความทนทาน (Endurance) สูงมาก
    * กินไฟต่ำมาก (ไม่ต้องใช้ไฟเลี้ยงเพื่อคงสภาพข้อมูล)

### 2. Phase Change RAM (PRAM / PCRAM)
ใช้การเปลี่ยนสถานะทางกายภาพของวัสดุ (Chalcogenide glass)
* **กลไก:** สลับระหว่างสถานะ **Amorphous** (ความต้านทานสูง) และ **Crystalline** (ความต้านทานต่ำ) โดยใช้ความร้อน
* **จุดเด่น:** * ความหนาแน่นข้อมูลสูง
    * ตัวอย่างที่โดดเด่นคือเทคโนโลยี **3D XPoint**

### 3. Resistive RAM (ReRAM / RRAM)
ใช้การเปลี่ยนแปลง **ความต้านทานไฟฟ้า** ของวัสดุ Dielectric
* **กลไก:** สร้าง "สะพานนำไฟฟ้า" (Conductive Filament) ภายในชั้นฉนวน
* **จุดเด่น:** * โครงสร้างเล็กและเรียบง่าย
    * เหมาะสำหรับการทำ **3D Stacking** และงานด้าน AI (Neuromorphic Computing)

| **Nonvolatile (ข้อมูลไม่หาย)** | ❌ | ✅ | ✅ | ✅ | ✅ |

## 💡 สรุปภาพรวม
เทคโนโลยีเหล่านี้พยายามแก้ปัญหา **"Memory Wall"** เพื่อให้คอมพิวเตอร์สามารถประมวลผลได้รวดเร็วขึ้น ลดการใช้พลังงาน และรองรับคุณสมบัติ **Instant-On** (เปิดเครื่องแล้วพร้อมใช้งานทันทีโดยไม่ต้องโหลดข้อมูลจาก Disk)

> **Note:** ปัจจุบันยังติดข้อจำกัดเรื่องต้นทุนการผลิตที่สูงกว่า NAND Flash ทั่วไป จึงมักใช้ในงานเฉพาะทางหรือ Enterprise Server เป็นหลัก
